## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository about user account and presign upload.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development mode
$ yarn start:dev

# production mode
$ yarn start:prod
```
